Tag Name Category Description
ol list 'ordered list', hence the 'ol' tag, which is a numbered list
ul list 'unordered list', 'ul', a bulletted list.
li list a line item within a list
dl list description list - allows items with sub-items
dt list descriptive term part of descriptive list
dd list descriptive definition
table table defines the limits of a table element
thead table the header section of a table
tbody table the body of the table
tfoot table the footer section of a table
caption table the caption for a table element
tr table a row in a table. can be a child of the table itself or a child of a table subsection (thead, tbody, tfoot).
th table table header - sits at the top of a table column
td table table data - this is the actual data in the table's cell
